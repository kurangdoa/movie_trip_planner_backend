# Introduction

AI project to have agent that can find information about the stay that has the vibe on certain movie. For example: "I want my next trip to feel like Avatar. I'm looking for somewhere mystical with lots of neon blues and deep jungle vibes." the agent will answer such as "While I can't fly you to Pandora, I found a matching 'visual DNA' in this Eco-Dome in Tulum, Mexico."

# Tech Stack
- python: main language
- uv: python project
- docker: container
- makefile: local deployment
- chromadb: vector database
